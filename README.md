# 🚀 TCE Gen AI Course - Perceptron Hands-On Assignment

Welcome to the hands-on assignment for the Foundations of Deep Learning and Generative AI course. This repository contains the starter code for your Perceptron practical session.

---

## 🛠️ Step 1: Clone your Repository
Each student gets their own private repository copy. To clone it to your local machine:
```bash
git clone <your-repository-url>
```

---

## ☁️ Step 2: Run in Google Colab (Recommended)
If you don't have a local Python environment, you can run this assignment in **Google Colab**!

1. Open [Google Colab](https://colab.research.google.com).
2. Click on the **GitHub** tab.
3. Paste the URL of your repository (the one you are reading right now on GitHub).
4. Click on `hands-on/Perceptron_Hands_ON.ipynb` to open it.

---

## 🧪 Step 3: Complete the Exercises
Work through the notebook cells. You will verify AND gates, learn weights, and visualize decision boundaries in a step-by-step educational flow.

---

## 🗳️ Step 4: Auto-Submit Your Work
When you are done, run the **Auto-Submission** cell at the very bottom of the notebook! It will automatically push your completed notebook back to your GitHub Classroom repository for grading.

```python
# In your notebook (at the bottom):
submit_to_classroom()
```

No need to use the terminal for submission! 🥳
